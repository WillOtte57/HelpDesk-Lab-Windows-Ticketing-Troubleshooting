# Help Desk Lab – Windows Ticketing & Troubleshooting
 Overview: 

This project is a hands-on Help Desk lab designed to simulate real Tier 1 IT support work. The lab demonstrates troubleshooting, documentation, and customer-focused support using a Windows virtual machine.

Skills Demonstrated:

- Windows 10/11 troubleshooting
- Help Desk ticket documentation
- User account and password support
- Basic networking diagnostics (ipconfig, ping)
- Printer and service troubleshooting
- Software installation and permissions
- Disk cleanup and system maintenance
- Onboarding and offboarding processes
- Knowledge base article creation

Tools Used: 

- VirtualBox
- Windows 10/11 Virtual Machine
- Built-in Windows administration tools
- GitHub for documentation and version control

Project Structure:

- `tickets/` – Completed help desk tickets with troubleshooting steps
- `kb/` – Knowledge base articles for common user issues
- `checklists/` – Onboarding and offboarding procedures
- `commands/` – Common Help Desk command references
- `screenshots/` – Visual proof of troubleshooting work

Completed Tickets:

1. Computer running slow
2. No internet connection
3. Printer not printing
4. Software installation failed
5. Account locked / password reset
6. Low disk space

What This Lab Shows Recruiters:

- Ability to troubleshoot common end-user issues
- Clear documentation and communication
- Understanding of Tier 1 Help Desk workflows
- Security-aware approach to account and system access
- Willingness to learn and follow IT processes

How This Was Built:

A Windows virtual machine was used to intentionally simulate common Help Desk problems. Each issue was troubleshot, resolved, documented, and supported with screenshots as proof of hands-on work.
